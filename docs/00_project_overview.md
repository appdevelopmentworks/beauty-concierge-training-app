# Project Overview

## Project Name
beauty-concierge-training-app

## Purpose
SBC（湘南美容クリニック）コンシェルジュ職の新卒向け「入職後1カ月研修」にフィットした、
スマホ向け研修アプリを作成する。

本アプリは、すでに作成済みの「ビジネスマナーアプリ」とは別物として扱い、
ビジネスマナー以外の初月学習内容を、反復学習しやすい形で提供する。

## Background
SBCの公式採用ページでは、新卒社員向け入職研修として、
東京・豊洲の研修会場で、コンシェルジュとしての基礎知識や基礎業務を
1カ月にわたって学ぶ研修があると明記されている。

また、コンシェルジュの仕事として、
受付、ヒアリング、施術やメニューの案内、契約・会計、
電話対応、カルテ作成、受付業務などが紹介されている。

そのため本アプリは、これらの基礎知識・基礎業務を
スマホで学べるように設計する。

## Scope
今回のMVPでは、以下を学習対象とする。

- 受付フロー
- カウンセリング基礎
- 施術・メニュー基礎知識
- 契約・会計の基礎
- 電話・メール・問い合わせ対応
- クレーム初期対応
- 個人情報・コンプライアンス基礎
- SBCの理念・行動基準・目標意識の理解

※ ビジネスマナーは対象外（別アプリで作成済みのため）

## Product Concept
「初月研修の自習・復習をスマホで回せるSBCコンシェルジュ新人研修アプリ」

単なるクイズアプリではなく、
ケース問題・会話シミュレーション・フローチェックを含む
マイクロラーニング型の研修アプリとする。

## Target User
- SBCのコンシェルジュ新卒入職者
- 入社後1カ月の研修受講者
- 配属前・配属直後の初学者
- 研修担当者が補助教材として使うケース

## MVP Goal
入社初月に必要な基礎知識・基礎業務について、
スマホのみで以下ができる状態を目標とする。

1. 学習カテゴリを選べる
2. 1問ずつ学習できる
3. 正解・不正解と解説を確認できる
4. 会話シミュレーション型の問題を解ける
5. 学習履歴を端末内に保存できる
6. どのカテゴリをどこまで学習したか分かる

## Design Policy
- スマホファースト
- 片手操作しやすいUI
- 新人でも迷わないシンプル構成
- 余計な機能を入れすぎない
- 問題データはJSONで管理しやすくする
- 将来的にカテゴリ追加しやすい構成にする

## Out of Scope for MVP
- 管理者ログイン
- サーバーDB
- 本番運用レベルのユーザー管理
- 音声認識
- 動画配信
- 社内SSO連携
- 実際のSBC業務システムとの連携
